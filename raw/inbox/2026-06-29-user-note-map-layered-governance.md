# User Note: Map Knowledge Needs Layered Governance

- Date: 2026-06-29
- Type: knowledge governance / BP map model / local maintainer note

## Original Concern

The maintainer pointed out that the current Season 46 map profile page mixes stable map knowledge with seasonal rotation knowledge.

Maps themselves are relatively stable. A season changes which maps are in Ranked, and future patches can add or remove maps from the pool. Strong brawlers also change with new brawlers, balance updates, and meta shifts. Therefore map structure, season index, and brawler strength must be separated.

## Durable Principle

The knowledge base should separate:

- stable map entity pages: map geometry, map features, lanes, objective access, false-positive capabilities
- seasonal ranked pool indexes: which maps are currently active in a season
- patch/meta overlays: which brawlers are currently strong, weak, overpowered, or newly introduced
- brawler map-fit overlays: how a specific brawler's capabilities activate on specific map features

Stable map pages should not be rewritten just because a season changes. Seasonal pages should point to map pages. Meta pages and hero pages should overlay current strength and brawler-specific interactions.
