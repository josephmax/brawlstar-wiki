# User Note: Ban Pick Should Assume High-Level Play

- Date: 2026-06-29
- Type: player experience / BP strategy principle / local maintainer note
- Context: correction to a preliminary Power League Prodigy research summary

## Original Note Summary

The maintainer clarified that rank presets should not become a core Ban Pick strategy dimension for this knowledge base.

For low-rank matches, the practical path may be to use simple carrying approaches, such as forcing the strongest map pick, ignoring teammate influence, or picking an over-tuned brawler if left open. Those choices are useful for climbing, but they should be treated as shortcuts for noisy environments rather than as high-quality BP reasoning.

The durable BP model should assume both teammates and opponents are high-level. A strong draft is meant to guarantee the matchup's basic structure and reduce the weight of mechanical execution in the final result. If a player cannot execute the draft, that is a gameplay problem rather than a BP model problem.

## Durable Principle

- Formal BP analysis should optimize for high-level baseline play.
- Low-rank simplifications are tactical shortcuts, not strategic evaluation dimensions.
- Do not spend BP reasoning budget on modeling low-rank noise.
- Draft quality should be judged by whether it creates better win conditions, role coverage, counterplay, and execution burden under competent play.
