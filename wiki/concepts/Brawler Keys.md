# Brawler Keys

## 定义

`Brawler Keys` 是《Brawl Stars》中的解锁资源，用于在 `Brawl Pass Vault` 中兑换任意 Brawler。

## 核心作用

- 用钥匙直接兑换英雄
- 作为 Brawl Pass 的定向解锁奖励
- 替代部分 `Credits` 的英雄获取路径

## 主要来源

- `Brawl Pass` paid track
- `Brawl Pass Plus` track
- bonus tail rewards

## 系统位置

Brawler Keys 是赛季奖励系统里的英雄解锁入口，属于比普通资源更直接的定向兑换机制。

## 与其他概念的区别

- 不同于 `Credits`：Credits 是通用解锁资源，Brawler Keys 是 Vault 定向兑换资源
- 不同于 `Coins` / `Power Points`：它们用于升级，不用于解锁英雄
